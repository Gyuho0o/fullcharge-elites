# 완충 전우회 (The 100% Elites)

배터리가 100%인 사람들만 입장할 수 있는 엘리트 전용 익명 채팅 앱

## 컨셉

"현대인의 배터리 불안증을 유쾌한 선민의식으로 승화시킨다"

배터리 100%라는 조건으로 희소성을 만들고, 언제든 퇴장당할 수 있다는 긴장감과 함께 "나는 100% 엘리트다"라는 유쾌한 우월감을 제공합니다.

## 핵심 규칙

### 입장 조건
- 스마트폰 배터리가 **100%**일 때만 입장 가능
- 충전 중이든 아니든 상관없이 100%만 되면 OK

### 생존 규칙
- 입장 후 배터리를 사용하면서 채팅 가능
- 배터리가 **99%**로 떨어지면 **10초 카운트다운** 시작
- 카운트다운 중 **충전기 연결하면 생존** (카운트다운 취소)
- 10초 내 충전 안 하면 **퇴장(추방)**
- 퇴장 시 채팅방에 "OO님이 전우회를 배신했습니다" 메시지 표시

## 등급 시스템

채팅방 체류 시간에 따라 등급 상승:

| 등급 | 필요 시간 | 설명 | 채팅 버블 |
|------|-----------|------|-----------|
| 뉴비 | 0분 | 일단 들어오셨군요 | 기본 회색 |
| 정회원 | 5분 | 슬슬 감이 잡히시나요? | 블루그레이 |
| 터줏대감 | 30분 | 이제 충전기를 못 뺀다 | 네이비 |
| 전설 | 2시간 | 당신이 전설입니다 | 진한 네이비 |

- 등급에 따라 채팅 버블 색상이 다르게 표시됩니다
- 등급 상승 시 축하 애니메이션이 표시됩니다

## 주요 기능

### 게이트키퍼 화면
- 배터리 상태 시각적 표시
- 충전 중일 때 진행바 표시
- 100% 완충 시 입장 버튼 활성화
- 현재 접속자 수 실시간 표시

### 채팅 화면
- Firebase 기반 실시간 채팅
- 온라인 사용자 목록 및 랭킹
- 승급까지 남은 시간 카운트다운
- 입장 이후 메시지만 표시 (이전 메시지 안 보임)
- 메시지 전송 시 번개 이펙트

### 위험 모드 (99% 이하)
- 화면 가장자리 빨간 경고 효과
- 상단 경고 바에 카운트다운 표시
- "지금 충전하세요!" 안내

### 퇴장 화면
- 체류 기록 및 달성 등급 표시
- 전면 광고 표시 후 입장 화면으로 복귀

## 기술 스택

- **UI**: Kotlin + Jetpack Compose
- **실시간 채팅**: Firebase Realtime Database
- **배터리 모니터링**: Foreground Service
- **광고**: Google AdMob (전면 광고)
- **최소 SDK**: Android 8.0 (API 26)

## 설정 방법

1. Firebase 프로젝트 생성 후 `google-services.json` 파일을 `app/` 디렉토리에 추가
2. Firebase Realtime Database 활성화
3. AdMob 계정 설정 (선택사항)

자세한 설정은 [SETUP.md](docs/SETUP.md) 참고
