# 완충 전우회 (The 100% Elites)

## 컨셉

"현대인의 배터리 불안증을 유쾌한 선민의식으로 승화시킨다"는 비전을 가진 Android 앱이다. 배터리가 100%인 사람들만 입장할 수 있는 익명 채팅방으로, 일종의 엘리트 클럽 느낌을 준다.

## 입장 조건

- 스마트폰 배터리가 100%일 때만 입장 가능하다
- 충전 중이든 아니든 상관없이 100%만 되면 된다

## 채팅방 체류 규칙

- 입장 후 배터리를 사용하면서 채팅할 수 있다
- 배터리가 99%로 떨어지면 10초 카운트다운이 시작된다
- 카운트다운 중에 충전기를 연결하면 살아남을 수 있다 (카운트다운 취소)
- 10초 내에 충전하지 않으면 채팅방에서 퇴장(추방)된다

## 등급 시스템

채팅방에 체류한 시간에 따라 등급이 올라간다:

| 등급 | 필요 시간 | 설명 |
|------|-----------|------|
| 뉴비 | 0분 | 일단 들어오셨군요 |
| 정회원 | 5분 | 슬슬 감이 잡히시나요? |
| 터줏대감 | 30분 | 이제 충전기를 못 뺀다 |
| 전설 | 2시간 | 당신이 전설입니다 |

## UI/UX 특징

- **입장 화면 (게이트키퍼)**: 배터리 상태를 시각적으로 보여주고, 100%가 되면 입장 버튼 활성화
- **채팅 화면**: 실시간 채팅, 온라인 사용자 목록, 승급 카운트다운 표시
- **위험 모드**: 99%가 되면 화면에 빨간 경고 바와 테두리 효과가 나타나고 카운트다운 표시
- **퇴장 화면**: 체류 기록과 달성 등급을 보여주고, 전면 광고 표시 후 입장 화면으로 복귀

## 기술 스택

- Kotlin + Jetpack Compose로 UI 구현
- Firebase Realtime Database로 실시간 채팅 구현
- Foreground Service로 백그라운드 배터리 모니터링
- Google AdMob으로 퇴장 시 전면 광고 표시

## 핵심 가치

배터리 100%라는 조건으로 희소성을 만들고, 언제든 퇴장당할 수 있다는 긴장감과 함께 "나는 100% 엘리트다"라는 유쾌한 우월감을 제공하는 것이 이 앱의 재미 포인트다.
