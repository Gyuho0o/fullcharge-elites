개인정보처리방침

완충 전우회 (The 100% Elites)

시행일: 2024년 1월 1일

완충 전우회(이하 "앱")는 이용자의 개인정보를 중요시하며, 개인정보보호법을 준수합니다.

1. 수집하는 개인정보 항목

앱은 서비스 제공을 위해 최소한의 정보만 수집합니다.

자동 수집 항목

항목 수집 목적
기기 식별자 (익명) 사용자 구분 및 채팅 서비스 제공
배터리 상태 입장 자격 확인 및 게임 규칙 적용
채팅 메시지 실시간 채팅 서비스 제공
닉네임 채팅방 내 사용자 표시
접속 시간 등급 시스템 운영

광고 관련 수집 항목 (Google AdMob)

  • 광고 식별자
  • 기기 정보
  • 앱 사용 데이터

2. 개인정보의 이용 목적

수집된 정보는 다음 목적으로만 이용됩니다.

  • 채팅 서비스 제공
  • 배터리 기반 입장/퇴장 규칙 적용
  • 등급 시스템 운영
  • 맞춤형 광고 제공 (Google AdMob)
  • 서비스 개선 및 오류 수정

3. 개인정보의 보유 및 파기

항목 보유 기간
채팅 메시지 최근 50개만 보관, 이후 자동 삭제
접속 기록 채팅방 퇴장 시 삭제
닉네임 앱 삭제 시 삭제

앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다.

4. 개인정보의 제3자 제공

앱은 이용자의 개인정보를 제3자에게 제공하지 않습니다.

단, 다음의 경우는 예외입니다:

  • 법령에 의해 요구되는 경우
  • 이용자가 사전에 동의한 경우

5. 광고 및 분석 서비스

앱은 다음 서비스를 사용합니다:

Google AdMob

  • 목적: 광고 제공
  • 수집 정보: 광고 식별자, 기기 정보
  • 개인정보처리방침: https://policies.google.com/privacy

Firebase

  • 목적: 실시간 채팅, 서비스 분석
  • 수집 정보: 익명화된 사용 데이터
  • 개인정보처리방침: https://firebase.google.com/support/privacy

6. 이용자의 권리

이용자는 다음 권리를 행사할 수 있습니다:

  • 닉네임 변경: 앱 내에서 언제든지 변경 가능
  • 데이터 삭제: 앱 삭제 시 모든 로컬 데이터 삭제
  • 광고 개인화 거부: 기기 설정에서 광고 ID 재설정 또는 맞춤 광고 비활성화

7. 아동의 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만임이 확인된 경우 해당 정보를 즉시 삭제합니다.

8. 개인정보의 안전성 확보 조치

  • Firebase 보안 규칙을 통한 데이터 접근 제한
  • HTTPS를 통한 데이터 암호화 전송
  • 익명 식별자 사용으로 실제 개인정보 최소화

9. 개인정보처리방침의 변경

본 방침이 변경되는 경우 앱 업데이트 또는 공지를 통해 알려드립니다.
